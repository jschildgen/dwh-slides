Exercise Sheet 11 (2022-06-24/27/29)

Exercise 1: Window Functions

Write the queries for this exercise on the vaccinations table from the previous exercise sheet:
vaccinations(v_date, country, continent, cases, vaccinations, population)

  1. Show all vaccinations in Germany in January 2022. Calculate a rank for each row. The day with the highest number of vaccinations in Germany should get the rank 1.
  2. Show all vaccinations in Europe in January 2022. Again, calculate a rank. Now, rank 1 means that this was the day with the highest number of vaccinations in that country.
  3. Show all vaccinations in Germany. Display an additional column total_vaccinations that shows the total number of vaccinations until that day.
  4. For each country in Europe, show the total number of vaccinations and the ratio of that value to the overall number of vaccinations in Europe (e.g. Germany 17%, UK 14%, France 13%, ...). Write the query once without using the function RATIO_TO_REPORT and once with it.
  5. Show all vaccinations in Germany. Display an additional column "7_days_incidence". This is the number of new cases within 7 days per 100,000 inhabitants.
  6. For each country in Europe, calculate the 7-days incidence on the 1st May 2022.
