Exercise 1: Statistical Aggregation Functions, Skyline

In the next workbook (Challenge 4), you need to write down some analytical queries for your data warehouse. Pick at least one from this exercise sheet. The SQL queries should not just be constructed somehow, but also provide meaningful and relevant results.

  1. Write some queries in which you use statistical aggregation functions (e.g., STDDEV_POP).
  2. Write some queries in which you find out interesting correlations.
  3. Write some top-k queries (e.g., the 5 xxx with the highest yyy).
  4. Write some skyline queries (PREFERRING).
