Exercise 1: Statistical Aggregation Functions, Skyline

Write the queries for this exercise on the vaccinations table from the previous exercise sheet:
vaccinations(v_date, country, continent, cases, vaccinations, population)

  1. What was the average number of vaccinations per day in Germany in 2021? Also compute the standard deviation.
  2. Find out whether the number of corona cases within one day correlates with the number of vaccinations on that day.
  3. Find out whether the number of total corona cases within one country correlates with the total number of vaccinations within that country.
  4. For each country in Europe, find the day with the most vaccinations.
  5. Find the skyline of countries with a high vaccination number and a low number of total corona cases.
