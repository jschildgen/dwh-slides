Exercise Sheet 6 (2021-05-21)

Exercise 1: MongoDB

In this exercise sheet, you work on the same Pokémon dataset as on exercise sheet 2, but this time you should use MongoDB.

  1. Install MongoDB and find out how to import the Pokemón dataset from https://raw.githubusercontent.com/fanzeyi/pokemon.json/master/pokedex.json
  2. Find the english names of all monsters.
  3. What's the english name of the monster with id 115?
  4. What's the first type of the monster with the id 149?
  5. Use the aggregation pipeline to find the most frequently occurring monster type. Hint: the least frequently occurring type is Ice (only 34 monsters).
