Exercise 1: YAML-XML Converter

Develop a program in a programming language of your choice (e.g., Python) which opens a YAML file and produces an XML file with the same content. You can use the YAML file on lecture 2, slide 5 as an example.

  1. Use libraries to parse and work with YAML and XML data
  2. The XML document you produce will not have any attributes
  3. Transform a dictionary a: b ... into <a>b</a> elements with the dictionary key as its element name and value as its text content
  4. Transform a list a: - ... into an element <a><a>...</a><a>...</a></a> with a child element for each list entry.
