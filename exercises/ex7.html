Exercise Sheet 7 (2022-05-11/13/16)

Exercise 1: Schema Integration

For this exercise sheet, download the following two CSV files that contain german cities data:

  • ww-german-postal-codes.csv from https://www.wiwald.com/ds/kostenlose-liste-deutscher-postleitzahlen-und-zugehoeriger-orte/id/ww-german-postal-codes
  • zuordnung_plz_ort.csv from https://www.suche-postleitzahl.org/downloads#google_vignette

When you solve this exercise on your local database, create a new schema for this exercise: CREATE SCHEMA IF NOT EXISTS ex7; If you use the OTH database, use your FIRSTNAME_LASTNAME user and schema.

  1. Open the CSV files in a text editor (not in Excel!) to inspect their format.
  2. Prepare your staging area: For each CSV file, create a table in the Exasol database with the same attributes as in the CSV file.
  3. Which forms of heterogeneity exists between the two data sources? Give examples.
  4. Perform a schema mapping between the two schemas.
  5. Create an integrated schema in the Exasol database (use CREATE TABLE).

Exercise 2: IMPORT

  1. Write two IMPORT commands to import the CSV files in your staging area.
  2. For each of your two tables, how many rows were imported?
  3. How many cities are in the first dataset but not in the second, and vice versa?
