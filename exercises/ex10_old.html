Exercise Sheet 10 (2022-06-15/17/20)

Exercise 1: GROUPING SETS

  1. Download the CSV file "owid-covid-data.csv" from https://github.com/owid/covid-19-data/tree/master/public/data/ which contains Corona statistics (cases, vaccinations, ...) of all countries in the world. Inspect the CSV file in a text editor.
  2. Create a table on your Exasol database:
    create or replace table vaccinations (v_date date, country varchar(50), continent varchar(50), cases decimal(18,1), vaccinations decimal(18,1), population decimal(18,1));
  3. Use an IMPORT command to import the data from the CSV file. Hint: You can directly import the file from the GitHub server via HTTPS.
  4. Delete the following pre-aggregated rows: delete from vaccinations where continent is null;
  5. What's the (un)compressed sized of the vaccinations table?
  6. Take a look in a sample of the imported data to verify whether you imported the correct numbers. E.g., the number of cases and the number of vaccinations on the 1st May 2022 in Germany should be around 4000.
  7. Show total number of vaccinations for each country.
  8. For each country, how many vaccinations are there per 100,000 inhabitants? Show the top 10 with the highest vaccination rate.
  9. For each country, continent, and the whole world, show the number of vaccinations and the number of vaccinations per 100 inhabitants.
  10. For each day in 2022, and each month in 2022, show the total number of vaccinations.
  11. For each country, each year, each country and year, and the grand total, the the number of vaccinations.
  12. Show the total number of vaccinations, the number of vaccinations per year, per month, and per day.
