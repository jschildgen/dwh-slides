In the next workbook (Challenge 4), you need to write down some analytical queries for your data warehouse. The SQL queries should not just be constructed somehow, but also provide meaningful and relevant results.

Exercise 1: GROUPING SETS

  1. Write some queries in which you use GROUP BY and aggregation functions.
  2. Write some queries in which you use GROUPING SETS, ROLLUP, or CUBE.

Exercise 2: Window Functions

  1. Write some queries in which you use a ranking function (RANK, DENSE_RANK, or ROW_NUMBER).
  2. Write some queries in which you use windows functions (AVG|SUM|...(...) OVER (...)).

Exercise 3: Statistical Aggregation Functions

  1. Write some queries in which you use statistical aggregation functions (e.g., STDDEV_POP).
  2. Write some queries in which you find out interesting correlations.

Exercise 4: Skyline

  1. Write some top-k queries (e.g., the 5 xxx with the highest yyy).
  2. Write some skyline queries (PREFERRING).
